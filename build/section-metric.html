Section 4 Metric Spaces

¶
Definition 4.1.

Let \(d:X^2\to[0,\infty)\) be a function satisfying the following for all \(x,y,z\in X\text{.}\)

  1. \(d(x,y)=0\) if and only if \(x=y\text{.}\)
  2. \(d(x,y)=d(y,x)\)
  3. \(d(x,z)\leq d(x,y)+d(y,z)\)

Then \(d\) is said to be a metric on the set \(X\text{,}\) and

\begin{equation*} B_r(x)=\setBuilder{y\in X}{d(x,y)\lt r} \end{equation*}

is said to be a metric ball around \(x\).

Example 4.2. Examples of metrics.

Verify that each of the following is a metric.

  1. \(d(x,y)=1\) for all distinct \(x,y\in X\text{,}\) and \(d(x,x)=0\)
  2. \(d(x,y)=|y-x|\) for all \(x,y\in\mb R\)
  3. \(d(\tuple{x_0,x_1},\tuple{y_0,y_1})=\sqrt{(y_1-y_0)^2+(x_1-x_0)^2}\) for all \(\tuple{x_0,x_1},\tuple{y_0,y_1}\in\mb R^2\text{.}\)
  4. \(d(\tuple{x_0,x_1},\tuple{y_0,y_1})=|y_1-y_0|+|x_1-x_0|\) for all \(\tuple{x_0,x_1},\tuple{y_0,y_1}\in\mb R^2\text{.}\)
  5. \(d(\tuple{x_0,x_1},\tuple{y_0,y_1})=\max\setList{|y_1-y_0|,|x_1-x_0|}\) for all \(\tuple{x_0,x_1},\tuple{y_0,y_1}\in\mb R^2\text{.}\)
Theorem 4.3.

Let \(d\) be a metric on a set \(X\text{.}\) Then

\begin{equation*} \mc B=\setBuilder{B_r(x)}{x\in X,r>0} \end{equation*}

is a basis for a topology on \(X\text{.}\)

Definition 4.4.

The topology generated by the basis given in Theorem 4.3 is called the topology generated by the metric.

A given topology is said to be metrizable if there exists some metric that generates it. Two metrics are said to be topologically equivalent if they generate the same topology.

Proposition 4.5.

Every discrete space is metrizable.

Theorem 4.6.

Every metrizable space is \(T_4\text{.}\)

Theorem 4.7.

Two bases \(\mc B_0,\mc B_1\) generate the same topology if and only if for all \(x\in B_0\in\mc B_0\) there exists \(B_1\in\mc B_1\) such that \(x\in B_1\subseteq B_0\text{,}\) and for all \(x\in B_1\in\mc B_1\) there exists \(B_0\in\mc B_0\) such that \(x\in B_0\subseteq B_1\text{.}\)

Theorem 4.8.

Let \(B_{0,r}(x),B_{1,r}(x)\) be the metric balls around \(x\) given by two metrics \(d_0,d_1\) respectively. Then \(d_0,d_1\) are topologically equivalent if and only if for all \(x\in X\) and \(\epsilon\gt 0\text{,}\) there exists \(\delta\gt 0\) such that \(B_{0,\delta}(x)\subseteq B_{1,\epsilon}\) and \(B_{1,\delta}(x)\subseteq B_{0,\epsilon}(x)\text{.}\)

Definition 4.9.

For \(\vec x=\tuple{x_0,\dots,x_{n-1}}\in\mb R^n\text{,}\) let \(\vec x(i)=x_i\text{.}\)

Theorem 4.10.

The following metrics on \(\mb R^n\) are topologically equivalent.

  1. \(d(\vec x,\vec y)=\sqrt{\sum_{0\leq i\lt n}(\vec y(i)-\vec x(i))^2}\)
  2. \(d(\vec x,\vec y)=\sum_{0\leq i\lt n}|\vec y(i)-\vec x(i)|\)
  3. \(d(\vec x,\vec y)=\max\setBuilder{|\vec y(i)-\vec x(i)|}{0\leq i\lt n}\)
Definition 4.11.

The topology generated by the metrics given in Theorem 4.10 is called the Euclidean topology on \(\mb R^n\text{.}\)

Definition 4.12.

A local basis at a point \(x\) is a collection of open sets \(\mc B_x\) such that for every neighborhood \(U\) of \(x\text{,}\) there exists \(B\in\mc B_x\) such that \(x\in B_x\subseteq U\text{.}\)

Definition 4.13.

A space is said to be first-countable if there exists a countable local basis at every point of the space.

A space is said to be second-countable if there exists a countable basis for the space.

Proposition 4.14.

Every second-countable space is first-countable.

Proposition 4.15.

Every metrizable space is first-countable

Definition 4.16.

A space is said to be separable if there exists a countable dense subset of the space.

Theorem 4.17.

Let \(X\) be metrizable. Then \(X\) is second-countable if and only if it is separable.

Proposition 4.18.

Every Euclidean space is separable and second-countable.

Theorem 4.19.

For \(\vec x,\vec y\in\mb R^2\text{,}\) let \(d(\vec x,\vec y)=1\) if \(\vec x(1)\not=\vec y(1)\text{,}\) and \(d(\vec x,\vec y)=|\vec y(0)-\vec x(0)|\) otherwise. Then \(d\) is a metric generating a non-separable, non-discrete topology on \(\mb R^2\text{.}\)

Theorem 4.20.

The subspace \(\setBuilder{\vec x}{\vec{x}(1)\in\setList{0,1}}\) of the space defined in Theorem 4.19 is homeomorphic to the subspace \((0,1)\cup(2,3)\) of the Euclidean line.

Definition 4.21.

A point \(x\) is called a sequential limit point of a set \(A\) iff there exists a countable subset \(B\subseteq A\setminus\setList{x}\) such that every neighborhood of \(x\) contains all but finitely many points of \(B\text{.}\)

Proposition 4.22.

Every sequential limit point of a set is a limit point of that set.

Theorem 4.23.

Let \(X\) be first-countable. Then \(x\) is a limit point of a set if and only if \(x\) is a sequental limit point of that set.

Definition 4.24.

A Cauchy sequence is a countably infinte set \(A\) such that for all \(\epsilon\gt0\text{,}\) the set \(\setBuilder{x\in A}{\exists y\in A(d(x,y)\geq\epsilon)}\) is finite.

Definition 4.25.

A complete metric is a metric such that every Cauchy sequence has a sequential limit point.

A topology that can be generated by a complete metric is said to be completely metrizable.

Proposition 4.26.

Every Euclidean space is completely metrizable.

Proposition 4.27.

Let \(d:X\to[0,\infty)\) be a metric and \(Y\subseteq X\text{.}\) Then \(d\) restricted to \(Y\) generates the subspace topology on \(Y\text{.}\) (Therefore, every subspace of a metrizable space is metrizable.)

Theorem 4.28.

The subspace \((0,1)\) of the Euclidean line is completely metrizable, but not by the topology inherited from \(\mb R\text{.}\)

Theorem 4.29.

The subspace \(\mb Q\) of the Euclidean line is metrizable, but not completely metrizable.

Theorem 4.30.

The subspace \(\mb R\setminus\mb Q\) of the Euclidean line is completely metrizable, but not by the topology inherited from \(\mb R\text{.}\)

Theorem 4.31.

Metrizable and completely metrizable are topological properties. That is, if \(X\) and \(Y\) are homeomorphic, then \(X\) is (completely) metrizable if and only if \(Y\) is too.
