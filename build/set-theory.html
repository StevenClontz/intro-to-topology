A review of basic results/definitions concerning sets and the reals.

Definition A.1
  • \(\mb R\) is the set of real numbers.
  • \(\mb Z\) is the set of integers.
  • \(\mb N=\setBuilder{z\in\mb Z}{z\geq 0}=\setList{0,1,2,\dots}\) is the set of natural numbers, which includes zero.
  • \(\mb Q=\setBuilder{\frac{z}{n+1}}{z\in\mb Z,n\in\mb N}\) is the set of rational numbers.
Theorem A.2

The Archemedian Property of the real numbers guarantees that for each positive real number \(x>0\text{,}\) there exists a natural number \(n\in\mb N\) such that \(\frac{1}{n}\lt x\text{.}\)

Theorem A.3

De Morgan's Laws: Let \(\mc A\) be a collection of subsets of \(X\text{.}\)

\begin{equation*} X\setminus\bigcup_{A\in\mc A}A=\bigcap_{A\in\mc A}(X\setminus A) \end{equation*}
\begin{equation*} X\setminus\bigcap_{A\in\mc A}A=\bigcup_{A\in\mc A}(X\setminus A) \end{equation*}
Theorem A.4

Let \(S\subseteq \mb R\) be a set of real numbers with a lower bound. Then there exists a greatest lower bound (a.k.a. infimum) \(\glb S=\inf S\text{.}\)

Let \(S\subseteq \mb R\) be a set of real numbers with a lower bound. Then there exists a least upper bound (a.k.a. supremum) \(\lub S=\sup S\text{.}\)
