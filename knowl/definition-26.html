Definition 3.1.

A set is said to be finite if there exists an injection from that set to the set \(\setList{0,1,\dots,n-1}\) for some \(n\in\mathbb N\text{.}\) A set is infinite if it is not finite.

A set is said to be countable if there exists an injection from that set to the set \(\mb N\text{.}\) A set is uncountable if it is not countable.
